

我在windows上一个数据库ui软件上，如何连接到数据库？

● 最简单的方式是用 SSH 隧道，大多数数据库 GUI 软件都内置支持。

  步骤

  1. 先在服务器上启动端口转发（只需一次）：

  ssh <用户名>@<服务器IP>

  # 在服务器上运行
  docker run -d --name pg-tunnel --rm \
    --network ho_backend \
    -p 127.0.0.1:15432:5432 \
    alpine/socat \
    TCP-LISTEN:5432,fork,reuseaddr TCP:pgbouncer-rw:5432

  2. 在 GUI 软件中配置连接（以 DBeaver 为例）：

  SSH 标签页：

  ┌────────────────┬─────────────────────────┐
  │      参数      │           值            │
  ├────────────────┼─────────────────────────┤
  │ Use SSH Tunnel │ 勾选                    │
  ├────────────────┼─────────────────────────┤
  │ Host           │ 你的服务器 IP           │
  ├────────────────┼─────────────────────────┤
  │ Port           │ 22                      │
  ├────────────────┼─────────────────────────┤
  │ Username       │ SSH 用户名              │
  ├────────────────┼─────────────────────────┤
  │ Authentication │ Password 或 Private Key │
  └────────────────┴─────────────────────────┘

  Main / 连接标签页：

  ┌──────────┬────────────────────────┐
  │   参数   │           值           │
  ├──────────┼────────────────────────┤
  │ Host     │ 127.0.0.1              │
  ├──────────┼────────────────────────┤
  │ Port     │ 15432                  │
  ├──────────┼────────────────────────┤
  │ Database │ mydb                   │
  ├──────────┼────────────────────────┤
  │ Username │ user                   │
  ├──────────┼────────────────────────┤
  │ Password │ 你的 POSTGRES_PASSWORD │
  └──────────┴────────────────────────┘

  其他软件（DataGrip、Navicat、pgAdmin）操作类似，都有 SSH Tunnel 选项。

  3. 用完后在服务器上清理：

  docker stop pg-tunnel

  如果你的软件不支持 SSH 隧道

  在 Windows PowerShell 中手动建隧道：

  ssh -N -L 15432:127.0.0.1:15432 <用户名>@<服务器IP>

  保持这个窗口不关，然后软件直连 127.0.0.1:15432 即可（不需要配 SSH 标签页）。
