共计：18 业务容器 + 10 数据容器

2. 蓝绿发布策略

                 Caddy/LB
                /        \
         [Blue 当前]   [Green 新版]
          api-gw x2     api-gw x2
          user-svc x2   user-svc x2
          ...            ...

发布流程:
1. docker compose -f green.yml up  ← 启动绿组
2. 健康检查 /health 全部通过
3. Caddy upstream 切流至 Green
4. 观察 5min 指标正常
5. docker compose -f blue.yml down ← 销毁蓝组
6. 回滚: Caddy 切回 Blue（< 10s）

- 业务容器蓝绿切换，数据容器不参与蓝绿
- 通过 Docker label deploy=blue/green 区分组
- Caddy API 动态切换 upstream，零停机
