关键设计决策

1. Docker 网络：通过 *_SERVICE_URL 环境变量实现 Docker 容器间通信，本地开发自动 fallback 到 localhost:port
2. 启动顺序：depends_on + condition: service_healthy 确保 PG/Redis 先就绪，再启动业务服务
3. 所有服务共享 JWT secrets：因为 getConfig() 在所有服务导入 @repo/database 时会被触发（redis.ts
   模块级调用）
4. Caddy 开发环境关闭 HTTPS：auto_https off，生产环境移除即自动签发证书
