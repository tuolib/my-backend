│                                                                                                                                                                                                   │
│ 当前 API 入口层（Caddy）为单点部署在 S3 上，S3 宕机则全部服务不可用。数据层已有 HA（Patroni + Redis Sentinel），但入口层和编排层需要升级。迁移到 K8s                                              │

│                                                                                                                                                                                                   │
│ 用户选择： 自建机房 kubeadm + 全部迁入 K8s（含数据层）+ Helm Chart 编排                                                                                                                           │
